Fhfscreenadaptation

作为一个简单的布局文件，FHFScreenAdatation实现了简单布局，你只需要嵌入FHFScreenAdatation包，就只需要简单的设置控件在iphone5尺寸下的frame就能够实现所有屏幕的尺寸的布局效果完全一样,而且完全按照设计图给出的frame来赋值，不用一次计算.设计思想非常简单，就是根据其他屏幕与iphone5屏幕尺寸比来重新给出frame

View project on GitHub

FHFScreenAdaptation

实例：

UIButton *testBtn = [[UIButton alloc] initWithFrame:CGRectMakeEx(0, 100, 640, 300)];

640为4.0寸的两倍像素点宽度；这句话的效果可以让testBtn占整个屏幕的宽度，高度根据300适配 注意：CGRectMakeEx(0, 100, 640, 300)里面的值全是实际值得两倍 你会发现这些值完全是设计图上面给出的值。不用你在苦逼的除以2.。。。。。。。 ![iPhone6p]
